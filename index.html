Git Adventure

What is Git and why is it so important?

Git is by far, the most widely used version control system in the world today. Originally developed in 2005 by Linus Torvalds (also famous for creating the original Linux kernal). We use git to track changes that have been made to our code, and allow multiple people to work a project at the same time, without overwriting each other's changes.

What’s the working tree?

The working tree, or working directory, consists of files that you are currently working on. You can think of a working tree as a file system where you can view and modify files. The index, or staging area, is where commits are prepared. The index compares the files in the working tree to the files in the repo.

What’s the staging area?

The staging area is like a rough draft space, it's where you can git add the version of a file or multiple files that you want to save in your next commit (in other words in the next version of your project).
